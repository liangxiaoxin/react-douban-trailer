# react-douban-trailer
#### react 构建的一个豆瓣预告片网站 数据来自个人爬虫

第一步：数据来自另一个本地node项目，需运行douban-trailer-mock 项目

项目地址: [douban-trailer-mock](https://github.com/liangxiaoxin/douban-trailer-mock)

第二步：npm install

第三步：npm run start
